Rules
Read Carefully (Baca dengan seksama)
Rules #1

Users are not allowed to have more than 2 accounts in 1 device.

Pengguna tidak diperbolehkan memiliki lebih dari 2 akun dalam 1 perangkat.
Rules #2

Users are not allowed to use any Automation and Bot apps.

Pengguna tidak diizinkan untuk menggunakan aplikasi Otomatisasi dan Bot apa pun.
Rules #3

Users not allowed to use VPN and Proxy intentional (for User from Iran please disable your VPN when using this apps).

Pengguna tidak diizinkan menggunakan VPN dan Proxy secara disengaja. (Untuk pengguna dari Iran tolong matikan VPN untuk aplikasi ini).
Rules #4

User not allowed to use referral code within 1 device (in case of account more than 2 in 1 device)

Pengguna tidak diizinkan untuk menggunakan kode rujukan dalam 1 perangkat (jika akun lebih dari 2 dalam 1 perangkat)
Rules #5

Users are not allowed to block ad impressions.

Pengguna tidak diizinkan untuk memblokir tayangan iklan
